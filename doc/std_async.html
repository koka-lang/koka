Asynchronous primitives

This module is based closely on [1] and aims to have robust and composable asynchronous primitives. In particular, any outstanding asynchronous operation can be canceled (through cancel) within a certain scope which allows for composable primitives like timeoutstd/async/timeout: forall<a,e> (secs : duration, action : () -> <asyncx|e> a) -> <asyncx|e> maybe<a> and firstofstd/async/firstof: forall<a,e> (a : () -> <asyncx|e> a, b : () -> <asyncx|e> a) -> <asyncx|e> a.

References

[1]Daan Leijen. Structured Asynchrony with Algebraic Effects. Microsoft Research technical report MSR-TR-2017-21, May 2017. pdf 🔎
